# 上架前合规自查清单

## 使用方式

发布 listing 前，逐项核对以下清单。所有「✅」才可安全发布；任何「❌」需修改后重新核查。

---

## 通用检查项（所有平台）

### 知识产权

- [ ] 标题/描述中**没有**使用任何他人注册商标名称
- [ ] 没有使用「genuine / authentic / original + 他人品牌名」
- [ ] 如使用「Compatible with [品牌]」，已添加「Not manufactured by / not affiliated with [品牌]」声明
- [ ] 产品图片中没有显示他人品牌的 logo 或包装
- [ ] 描述内容为原创，未复制竞品 listing
- [ ] 如声称拥有专利，已确认实际持有或已提交申请

### 广告法合规

- [ ] 没有使用无法证明的绝对化表述（最好、最强、第一…）
- [ ] 没有无依据的数字声称（「比竞品好 X%」需有来源）
- [ ] 没有医疗/治疗/治愈/诊断类声称（非药品/医疗器械）
- [ ] 保健品/膳食补充剂已附加 FDA 免责声明
- [ ] 没有虚假原价再打折的价格欺诈
- [ ] 如引用认证（CE/FCC/UL），已确认实际持有证书

### 平台联系/评价规定

- [ ] 没有任何联系方式（电话/邮箱/网址/微信）
- [ ] 没有外部链接
- [ ] 没有引导留好评或要求删差评的语句
- [ ] 没有承诺返现换评

---

## Amazon 专项检查

### 标题
- [ ] 字符数在品类上限内（大多数品类 ≤ 200）
- [ ] 没有促销词：Sale / Free / Discount / Best Seller / Amazing
- [ ] 没有特殊符号：! ? * @ # $ % ^ & ( )
- [ ] 没有全大写的非品牌/缩略词内容
- [ ] 没有价格信息

### 五点描述
- [ ] 没有 URL 或联系方式
- [ ] 没有促销信息（Buy 2 get 1 / Limited offer）
- [ ] 没有 HTML 标签（五点描述不支持）
- [ ] 每条不超过 1000 字符

### 产品描述
- [ ] 如使用 HTML，只使用支持的标签（`<b>` `<br>` `<ul>` `<li>` `<p>`）
- [ ] 没有竞品平台名称（eBay / Walmart / Shopify）
- [ ] 没有外部购物链接

### 主图
- [ ] 背景为纯白（RGB 255,255,255）
- [ ] 产品占图面积 ≥ 85%
- [ ] 没有水印、文字覆盖、边框、促销标签

### Search Terms（后台）
- [ ] 总字节数 ≤ 250
- [ ] 没有竞品商标词
- [ ] 没有重复词（同一词不出现两次）
- [ ] 词间用空格分隔（不用逗号）

---

## eBay 专项检查

- [ ] 标题 ≤ 80 字符
- [ ] 标题开头没有 ! * $ 等特殊符号
- [ ] 没有「LOOK」「WOW」「L@@K」等注意力词
- [ ] 商品状态与实际一致（New / Used / Refurbished）
- [ ] 没有嵌入其他电商平台购买链接

---

## Shopee/Lazada 专项检查

- [ ] 没有竞品平台名称（Lazada/Shopee 互相禁用）
- [ ] 没有引导线下交易的内容
- [ ] 没有政治/宗教敏感内容
- [ ] 品类特殊规定已核查（印尼/马来西亚清真，泰国王室等）

---

## 跨境合规检查

### 认证类
- [ ] 电器产品已核查目标市场认证要求（CE/FCC/PSE/KC/RCM）
- [ ] 儿童产品已核查 CPSC/EN71 要求
- [ ] 锂电池已标注真实容量（mAh/Wh）和航空限制

### 标签类
- [ ] 纺织品已标注材质成分和原产国（美国 FTC）
- [ ] 儿童产品已标注适用年龄警告
- [ ] 声称原产国与实际生产地一致

### 特殊品类
- [ ] 无人机已确认目标市场登记要求
- [ ] 激光产品已标注正确级别
- [ ] 监控/摄像产品已明确告知录像功能

---

## 中国市场特别检查（天猫/京东/拼多多/国内平台）

- [ ] 没有中国广告法极限词（最、第一、唯一、顶级…）
- [ ] 没有未经授权使用国家/政府背书
- [ ] 保健品没有超出批准功能范围的声称
- [ ] 没有医疗器械声称（非医疗器械产品）
- [ ] 没有违反社会公德的内容
