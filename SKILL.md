# Skill: 跨境电商内容合规审查

## 概述

对跨境电商平台的产品标题、五点描述、产品描述、图片文案、关键词等内容进行全面合规风险扫描。覆盖：知识产权侵权、广告违规、平台政策违规、跨境法规风险四大维度。

输出结构化审查报告，包含风险等级、逐条问题定位、修改建议及合规重写版本。

---

## 触发方式

用户提供以下任意内容时启动审查：
- 产品标题 / 五点描述 / 产品描述
- 图片 Alt 文字 / Banner 广告文案
- Search Terms / 后台关键词
- 店铺介绍 / 品牌故事文案
- 用户提问：「这个描述有没有问题」「这样写会不会被下架」

若用户未指定平台，默认按 **Amazon 美国站** 标准审查，并在报告中注明。

## 工作流程

### Step 1 · 内容接收与分类

接收用户提交的内容后，识别以下信息：

| 字段 | 说明 | 默认值 |
|------|------|-------|
| 内容类型 | 标题 / 五点描述 / 产品描述 / 图文广告 / 关键词 / 混合 | 自动识别 |
| 目标平台 | Amazon / eBay / Shopee / Lazada / Shopify / Temu / Walmart | Amazon |
| 目标市场 | 美国 / 英国 / 德国 / 欧盟 / 东南亚 / 日本 / 中东 | 美国 |
| 产品品类 | 电子 / 服装 / 家居 / 美容 / 食品 / 玩具 / 健康 / 其他 | 自动识别 |

### Step 2 · 四维度扫描

按以下顺序逐一扫描，详细规则见各 references 文件：

#### 维度 A：知识产权风险
> 详见 `references/ip-trademark.md`

- [ ] 标题/描述中是否直接使用他人注册商标名称
- [ ] 是否使用「compatible with」「for xxx」暗示正品关联
- [ ] 是否使用知名品牌词作为关键词修饰语
- [ ] 产品图片是否包含他人 logo / 包装
- [ ] 是否复制竞品描述（构成版权侵权）
- [ ] 专利描述风险：是否声称持有专利而实际未注册

**常见侵权词汇模式：**
```
Apple / iPhone / Samsung / Nike / Adidas / Disney / Marvel / [任何注册商标]
AirPods style / iPhone-like / Nike inspired
Genuine / Original / Authentic [+他人品牌名]
```

#### 维度 B：广告违规风险
> 详见 `references/advertising-law.md`

**B1 · 绝对化用语（各市场广告法均禁止）**
```
最好 / 最优 / 最强 / 第一 / 顶级 / 无与伦比
Best / #1 / Top / Greatest / World's Best / Unbeatable
Beste / Größte / Einzigartig（德语）
Meilleur / Premier / Inégalé（法语）
```

**B2 · 虚假宣传**
- 功效无法证明的声称（「治疗」「治愈」「医疗级」）
- 数据无来源的对比（「比竞品好 50%」）
- 无认证的认证声称（「FDA 认证」实际未认证）
- 夸大用户评价（「100% 满意」「零差评」）

**B3 · 医疗/健康类违规声称**
- 任何「治疗」「诊断」「预防疾病」的表述
- 未经批准的保健功效声称
- 适用：美容仪 / 保健品 / 食品 / 电子烟 / 宠物食品

**B4 · 价格欺诈类**
- 虚构原价再打折
- 「限时」「仅剩 X 件」制造紧迫感（实际常备库存）
- Amazon 明确禁止在 listing 中出现价格信息

#### 维度 C：平台政策违规
> 详见 `references/platform-policies.md`

**C1 · 通用平台禁止内容**
```
联系方式（电话/邮箱/网址/微信/WhatsApp）
外部链接
促销价格、折扣信息
「Buy 2 Get 1」等促销话术（listing 正文）
竞品品牌名（特指负面对比）
```

**C2 · Amazon 专项规则**
- 标题禁用：! @ # $ % ^ & * ( ) 等符号（品牌名除外）
- 标题禁用词：Sale / Free / Discount / Cheap / Best Seller / Amazing / Guaranteed
- 五点描述：不得包含公司信息、网址、促销
- 图片：主图必须白底，不得含水印/文字/边框
- Search Terms：不得含竞品品牌词、重复词、无关词

**C3 · eBay 专项规则**
- 标题禁用：!, *, $（开头），LOOK, WOW 等注意词
- 不得在描述中嵌入外部购物链接

**C4 · Shopee/Lazada 专项规则**
- 不得承诺平台以外的交易（线下交易）
- 不得出现竞品平台名称（Lazada/Shopee 相互禁止）
- 禁止含有政治敏感内容

**C5 · 评论操控类**
- 要求买家留好评 / 承诺返现换评
- 「Leave us a 5-star review」等引导语
- 在包装卡片上要求删除差评

#### 维度 D：跨境法规风险
> 详见 `references/cross-border-regulations.md`

**D1 · 产品安全声称**
- 电器类：是否错误声称认证（CE / FCC / RoHS / PSE / KC）
- 儿童产品：是否违反 CPSC / EN71 安全标准声称
- 食品接触材料：是否错误使用「食品级」「BPA Free」

**D2 · 进出口管制**
- 加密/通信类产品：美国 EAR 出口管制声称
- 军事/双用途产品：ITAR 合规
- 濒危物种材料（CITES）：不得声称含相关材料

**D3 · 隐私与数据**
- 智能设备/摄像头：是否有隐私数据收集声称
- GDPR（欧盟）：数据处理声称
- CCPA（加州）：适用于在加州销售

**D4 · 特殊品类监管**
- 保健食品 / 膳食补充剂：FDA DSHEA 合规
- 医疗器械：510(k) / Class II 声称
- 儿童玩具：ASTM F963 / EN71 声称
- 电池产品：UN38.3 / IATA 声称

### Step 3 · 风险评级

对每个发现的问题独立评级，并给出整体评级：

| 等级 | 标识 | 定义 | 典型后果 |
|------|------|------|---------|
| 严重 | 🔴 HIGH | 直接违规，存在立即被下架/封号/法律诉讼风险 | 强制下架、账号暂停、法律索赔 |
| 中等 | 🟡 MEDIUM | 违反平台政策或广告法，存在被处罚的较高风险 | 警告、listing 被压权重、移除 |
| 低等 | 🟢 LOW | 存在潜在风险，建议优化但短期影响有限 | 影响 SEO 排名、转化率 |
| 提示 | 🔵 INFO | 最佳实践建议，不涉及违规 | 无处罚风险，仅影响效果 |

**整体评级逻辑：**
- 有任意 1 个 HIGH → 整体 **严重**
- 无 HIGH，有 ≥ 2 个 MEDIUM → 整体 **中等偏高**
- 无 HIGH，有 1 个 MEDIUM → 整体 **中等**
- 仅 LOW/INFO → 整体 **低风险**
- 无任何问题 → 整体 **合规通过**

### Step 4 · 生成报告

按 `assets/report-template.md` 格式输出，包含：
1. 整体评级与摘要
2. 逐条问题清单（定位原文 + 风险说明 + 修改建议）
3. 合规重写版本（自动生成修改后的文案）
4. 后续行动建议

## 特殊品类处理规则

### 保健食品 / 膳食补充剂
- 禁止：「治疗」「预防疾病」「治愈」任何疾病名称
- 允许：「有助于支持免疫系统」（结构功能声称，需 FDA 备案）
- 必须包含：「These statements have not been evaluated by the FDA...」免责声明

### 医美 / 美容仪器
- 禁止：「去皱」「提升」「嫩肤」的医疗声称
- 允许：「有助于皮肤看起来更紧致」（外观描述）
- 二类医疗器械：需 510(k) 批准才可声称医疗功效

### 儿童玩具
- 必须标注适用年龄
- 禁止：「绝对安全」「无任何风险」等绝对安全声称
- 必须正确引用安全认证（ASTM F963 / EN71 / CE）

### 锂电池产品
- 必须标注：电池容量（mAh/Wh）、型号
- 航空运输限制需在描述中注明（UN38.3）
- 禁止虚报电池容量参数

### 电子烟 / 雾化产品
- Amazon / Shopee / 大多数平台**明确禁售**
- 任何与烟草相关的产品均触发高风险

## 注意事项

- **不提供法律意见**：本技能提供合规风险提示，不构成正式法律建议。高风险情况建议咨询专业跨境电商律师。
- **平台规则动态更新**：各平台政策持续变化，审查结论以平台最新官方政策为准。参考文件更新日期：2025 年。
- **地区差异**：同一内容在不同市场合规性可能不同，默认按所选市场审查，跨市场销售时需分别审查。
- 详细规则库见各 `references/` 文件。
