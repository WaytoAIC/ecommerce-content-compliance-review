# 各平台政策违规规则库

## Amazon

### 产品标题规范

**禁止内容（直接违规 → 🔴 HIGH）：**
```
促销词：        Sale / Free / Discount / Cheap / Lowest Price / Best Price
主观词：        Amazing / Perfect / Excellent / Wonderful / Incredible
排名词：        Best Seller / #1 / Top Rated / Most Popular / Award Winning
保证词：        Guaranteed / Warranty（标题中）/ Certified（无依据）
联系信息：      任何电话/邮件/网址/社交媒体账号
价格信息：      $XX.XX / XX% Off / Buy X Get Y
```

**符号限制（标题中禁用 → 🟡 MEDIUM）：**
```
禁用：  ! ? * @ # $ % ^ & ( ) + = { } | \ < > ~ ` '
允许：  , . - / 冒号: 分号;（用于分隔信息）
例外：  品牌名本身含特殊符号时（如 AT&T）允许保留
```

**全大写限制（🟢 LOW）：**
- 禁止：整个标题全大写
- 允许：品牌名全大写（SONY / ANKER）、缩略词（USB / LED / ANC）

---

### 产品图片规范

**主图（Main Image）强制要求：**
- 背景：纯白（RGB 255, 255, 255）
- 产品占比：≥ 85% 的图片面积
- 格式：JPEG / PNG / GIF，最小 1000×1000px

**主图禁止内容（🔴 HIGH）：**
```
- 水印、文字覆盖、标注箭头
- 促销标签（Sale / New / Hot）
- 品牌 logo 覆盖整图
- 人体模特（部分品类例外）
- 多产品拼接（变体主图）
- 边框、阴影背景
```

**副图（Additional Images）禁止内容（🟡 MEDIUM）：**
```
- 竞品品牌名称出现在对比图中
- 虚假数据对比
- 「Award / #1 / Best」等未经证实的标签
```

---

### 产品描述与五点描述

**五点描述禁止内容（🔴 HIGH）：**
```
- 任何 URL / 网站地址
- 公司联系方式（电话/邮箱）
- 促销或定价信息（Buy 2 get 1 / Limited time offer）
- 保修承诺（需转移到售后政策页）
- 重复标题内容（超过 50%）
```

**五点描述禁止内容（🟡 MEDIUM）：**
```
- 「See description below」类引导词
- HTML 标签（五点描述不支持 HTML）
- 全大写句子（整句大写）
```

**产品描述禁止内容（🔴 HIGH）：**
```
- 外部网站链接
- 其他平台名称（eBay / Walmart / Shopify）
- 第三方卖家评价截图描述
- 要求留评的语句
```

---

### Search Terms（后台关键词）

**严格禁止（🔴 HIGH）：**
```
- 竞品品牌名（Nike / Apple / Samsung 等注册商标）
- ASIN 号
- 亵渎语言、仇恨词汇
- 虚假声称（organic / natural 未经认证）
```

**技术性禁止（🟡 MEDIUM）：**
```
- 重复词汇（同一词出现多次）
- 标点符号（逗号、连字符不计入分隔，直接用空格）
- 超过 250 字节总限制
- 与标题/五点描述完全重复的词
```

---

### Amazon 评论政策

**严格禁止（🔴 HIGH，可导致账号暂停）：**
```
- 产品包装内附「留好评返现」卡片
- 邮件要求删除/修改差评
- 文案中包含「Please leave a 5-star review」
- 索要评论换取折扣或免费产品
- 在 Follow-up 邮件中直接要求五星评价
```

**允许的评论引导（合规）：**
```
- 「If you have any questions, please contact us first」
- 「Your feedback means a lot to us」（不指定好评）
- Vine 计划内产品寄送
```

## eBay

### 标题规范

**禁止（🔴 HIGH）：**
```
- 外部网站 URL
- 竞争卖家价格比较（「比 Amazon 便宜」）
- 假冒品牌声称
```

**禁止（🟡 MEDIUM）：**
```
开头禁用的符号：  !, *, $, ~, ?, #
注意力词汇：      LOOK, WOW, L@@K, !!!, ???
夸大词：         RARE（常见品使用）, UNIQUE（非独特品）
```

### 描述规范

**禁止（🔴 HIGH）：**
```
- 嵌入其他电商平台的购买链接
- 要求买家在 eBay 平台外完成交易
- 虚假商品状态（New 标签用于二手品）
```

**禁止（🟡 MEDIUM）：**
```
- 过度使用 HTML 特效（闪烁文字/音乐自动播放）
- 隐藏文字（白底白字关键词堆砌）
- 夸大「几乎全新」「完美品相」对明显有损耗的产品
```

## Shopee

### 内容禁区（🔴 HIGH）

```
政治敏感：      涉及台湾/西藏/香港政治立场的表述
宗教敏感：      不当使用宗教符号/名称
性暗示：        在非成人品类中的性暗示内容
竞品平台名：    Lazada / JD / Tokopedia 等竞品平台名称
线下交易引导：  提供联系方式引导平台外成交
```

### 内容禁区（🟡 MEDIUM）

```
绝对化用语：    最好/最便宜/第一（当地广告法禁止）
无依据认证：    「通过 XXX 认证」（无证据支撑）
```

```
- Shopee / JD / Tokopedia 等竞品平台名称
- 性暗示或成人内容（非成人品类）
- 线下交易引导

```

```
- 错误品类标注
```

## Shopify 独立站

独立站无统一内容审核，但以下法规仍适用：

**必须合规的法规（🔴 HIGH）：**
- 目标市场广告法（FTC / ASA / German UWG 等）
- 产品安全法规
- 隐私政策（GDPR / CCPA）

**常见违规（🟡 MEDIUM）：**
```
- 无隐私政策页面（GDPR 强制）
- 无退款/退货政策（EU 消费者权益法强制）
- 虚假折扣原价（FTC 价格标注指南）
- Cookie 弹窗缺失（EU ePrivacy）
```

## Temu

### 特别注意（全托管模式）

Temu 全托管卖家基本无法控制前端文案，但供货时需确保：

**供货资质合规（🔴 HIGH）：**
```
- 不得提供无 CE / FCC 认证的电器产品
- 不得提供侵权产品（商标/版权）
- 不得提供危险品（无 UN38.3 认证的锂电池）
```

## Walmart Marketplace

### 禁止内容（🔴 HIGH）

```
- 标题中出现价格或促销词
- 假冒品牌声称
- 药品/医疗设备的未经批准声称
```

### 禁止内容（🟡 MEDIUM）

```
- 「Walmart Exclusive」（非官方认证的专属声称）

- 竞争平台名称（Amazon / eBay）
```

## 通用违规高频词库

以下词汇在绝大多数平台 listing 中均属违规或高风险：

### 促销类（标题/描述中禁用）
```
Sale / On Sale / Clearance / Discount / % Off / BOGO
Flash Sale / Limited Time / Today Only / Last Chance
Free Gift / Buy X Get Y / Bundle Deal（listing 正文）
```

### 联系引导类（全平台禁用）
```
Call us / Email us / WhatsApp / WeChat / Line
www. / .com / .cn / @（邮箱）/ Tel: / Contact:
Follow us on Instagram / Facebook / TikTok
```

### 评论操控类（全平台禁用）
```
Leave a 5-star review / Please rate us
If satisfied, leave feedback / Review us
We'll refund if you review / 好评返现
```
