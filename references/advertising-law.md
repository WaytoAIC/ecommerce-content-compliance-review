# 各市场广告法合规规则库

## 一、美国广告法（FTC）

### 核心原则

美国联邦贸易委员会（FTC）要求广告内容：
1. **真实性**：不得有实质性欺骗或误导
2. **可证实性**：客观声称必须有充分证据支撑
3. **公平性**：不得利用不公平或欺骗性行为

### 明确违规类型

#### 虚假广告（🔴 HIGH）

```
无依据的功效声称：
- 「减重 20 磅 guaranteed」
- 「7 天治愈皮肤问题」
- 「提升 IQ」「增强记忆力」（无临床证据）

虚构折扣价：
- 标注「原价 $99.99，现价 $19.99」（从未以 $99.99 销售过）
- 人为虚高「建议零售价」（MSRP）再打折

虚假用户评价：
- 刊登从未真实发生的好评
- 修改真实评价内容
```

#### 误导性声称（🟡 MEDIUM）

```
不完整比较：
- 「比同类产品省电 30%」（未说明比较对象）
- 「消费者首选品牌」（未说明调研方法/范围）

暗示性欺骗：
- 商品图片展示配件，但实际不含配件
- 「包含全套工具」图片中显示的工具并非全部包含

选择性真实：
- 「医生推荐」（1 位医生 vs 研究共识）
- 「临床证明有效」（小样本/资助研究）
```

### FTC 背书与证言规则（2023 更新）

**必须披露的情况：**
- 有偿评价/推荐（包括免费产品换评）
- 员工/关联方评价
- 非典型结果的证言（需附免责声明）

**合规示例：**
```
✅ 「[产品名]帮助我减重了 15 磅。个人结果因人而异。」
✅ 「[Disclosure: 我收到免费产品进行测评]」
❌ 「这个产品让我减了 15 磅！」（暗示典型结果）
```

## 二、中国广告法（适用于中文 listing / 国内平台）

### 绝对禁用词（🔴 HIGH，违法）

根据《中华人民共和国广告法》第9条，以下表达**明确违法**：

**极限词（禁止）：**
```
最、最佳、最优、最高、最大、最强、最低、最便宜
顶级、顶尖、第一、唯一、首选、首位
极致、终极、无与伦比、史上最
```

**权威性词汇（无依据时禁止）：**
```
国家级、国际级、全球领先
世界级、行业领先、业内第一
央视推荐、官方推荐（非真实授权）
```

**违禁用法示例：**
```
❌ 「最好的蓝牙耳机」
❌ 「全球第一品牌」
❌ 「史上最强降噪」
❌ 「顶级音质」

✅ 「优质蓝牙耳机」
✅ 「卓越音质体验」
✅ 「出色的主动降噪效果」
```

### 其他违规类型（🟡 MEDIUM）

```
使用国家机关名义：「国家推荐」「政府认可」（未经授权）
涉及社会公德：低俗/歧视性内容
虚假宣传用户口碑：「销量第一」（无数据支撑）
医疗声称（非医疗产品）：「改善睡眠障碍」「治疗某某病」
```

### 保健品广告特别规定

保健品广告不得出现：
```
- 任何疾病治疗声称
- 「药到病除」「无效退款」等保证语
- 消费者证言描述疾病症状改善
- 与药品进行比较
```

## 三、欧盟广告法规

### 欧盟不公平商业行为指令（UCPD）

**欺骗性商业行为（🔴 HIGH）：**
```
- 对产品特征的虚假陈述（材质、重量、产地等）
- 虚构促销价格（未曾以所谓「原价」销售）
- 宣传已售完的特价品吸引流量转化为高价品
- 误导性环保声称（Green Washing）
```

**绿色宣传（Green Claims）规范（2024 强化执行）：**
- 「生态友好」「可持续」「环保」需有具体数据支撑
- 「碳中和」「零排放」需经第三方认证
- 禁止无依据的「100% 可回收」声称

### 德国广告法（UWG）

德国执法尤为严格，常见违规：
```
绝对化用语：    „beste" / „billigste" / „einzigartige"（最好/最便宜/独一无二）
比较广告：      竞品对比需真实、可验证、客观（否则构成不正当竞争）
价格误导：      UVP（建议零售价）必须是真实参考价格
```

### 英国广告法（ASA/CAP）

```
「Clinically proven」：需有同行评审的临床研究
「Dermatologist tested」：需真实测试，并说明测试结论
环保声称：需符合 ASA Green Claims 指南
```

## 四、各市场健康/医疗声称规范

### 美国 FDA 规则

**结构功能声称（Structure/Function Claims）—— 合法（需备案）：**
```
✅ 「有助于支持免疫系统健康」
✅ 「有助于维持正常的消化功能」
✅ 「有助于钙的吸收」
```

**疾病声称（Disease Claims）—— 非 OTC 药品不得使用（🔴 HIGH）：**
```
❌ 「治疗感冒」「预防癌症」「降低血压」
❌ 「减少关节炎症状」「改善抑郁」
❌ 「治愈失眠」「帮助糖尿病患者控制血糖」
```

**必须附加的免责声明：**
```
「These statements have not been evaluated by the Food and Drug Administration. 
This product is not intended to diagnose, treat, cure, or prevent any disease.」
```

### 中国保健食品

保健食品允许声称的27种功能（需注册批准）：
增强免疫力、辅助降血脂、辅助降血糖、抗氧化、辅助改善记忆...

**非保健食品不得声称上述功能（🔴 HIGH）**

### 欧盟 EFSA 健康声称

- 食品/保健品健康声称需 EFSA 批准
- 获批声称列表：EC Regulation 432/2012
- 未批准的声称（即使真实）也不得在 EU 使用

## 五、各国特殊广告禁区

### 中国
```
禁止：含有封建迷信内容
禁止：使用「国家级」「最高级」「最佳」
禁止：妨碍社会公共秩序或违背社会良好风尚
敏感：政治相关（党政军形象）
```

### 中东（沙特/阿联酋）
```
禁止：与伊斯兰价值观相冲突的内容
禁止：男女不当接触的图片
限制：酒精、猪肉、赌博相关产品广告
敏感：批评当地政府或王室的内容
```

### 德国
```
禁止：纳粹标志/符号（刑法 §86a）
禁止：否认大屠杀的陈述
限制：比较广告须严格遵循 UWG
```

### 日本
```
禁止：「No.1」「最高」等最高级表达（景品表示法）
禁止：无依据的价格比较
限制：医药品类严格管制
```

## 六、快速合规自查词表

### 高风险词汇（出现即审查）

**英文：**
```
Best / Finest / Greatest / Perfect / Ultimate / World-class
#1 / Top / Number One / Leading
Proven / Guaranteed / Certified（无依据）
Cure / Treat / Diagnose / Prevent（非药品）
FDA Approved / CE Certified（无证书支撑时）
100% / Always / Never / Every（绝对化）
Scientifically proven / Clinically tested（无引用来源）
```

**中文：**
```
最、第一、唯一、顶级、极致、无与伦比
国家级、国际领先、全球第一
治疗、治愈、消除、根治（非药品类）
100%、绝对、永久、彻底（效果声称）
官方推荐、央视认证（未经真实授权）
```

### 相对安全的替代表达

| 违规表达 | 合规替代 |
|---------|---------|
| 最好的耳机 | 出色的耳机 / 高性能耳机 |
| 治疗失眠 | 有助于营造舒适的睡眠环境 |
| 100% 满意 | 我们致力于您的满意 |
| 全球第一品牌 | 深受用户信赖的品牌 |
| 临床证明有效 | 经过测试 / 多用户验证 |
| 永久防水 | IP68 防水等级（具体参数） |
