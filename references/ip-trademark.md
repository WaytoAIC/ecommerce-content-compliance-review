# 知识产权与商标侵权参考库

## 一、商标侵权基础

### 什么构成商标侵权

在跨境电商场景中，以下行为可能构成商标侵权：

| 行为类型 | 示例 | 风险等级 |
|---------|------|---------|
| 直接使用他人商标 | 标题写「Nike 运动鞋」（非正品） | 🔴 严重 |
| 仿冒商标标识 | 使用与 Nike Swoosh 高度相似的 logo | 🔴 严重 |
| 混淆性近似使用 | 品牌名「Niike」「N1ke」 | 🔴 严重 |
| 关键词竞价引流 | Amazon PPC 使用竞品商标词 | 🟡 中等（灰色地带） |
| Compatible with 声称 | 「Compatible with iPhone 15」 | 🟢 低（有条件允许）|
| 反向仿冒 | 去除正品 logo 贴自己品牌销售 | 🔴 严重 |

### 合法使用商标的情形（指示性合理使用）

以下情形在美国法律框架下通常合法：
- **配件/配件兼容性声称**：`Compatible with Apple iPhone 15`（需满足条件，见下）
- **比较广告**：在真实对比中提及竞品品牌
- **描述性使用**：描述产品本身功能时必要提及

**「Compatible with」使用条件（美国）：**
1. 被提及的商标仅用于识别兼容的产品型号
2. 不暗示商标权人认可该产品
3. 不使用比必要更显著的商标展示方式
4. 附注「Not manufactured by / Not affiliated with [品牌名]」

## 三、侵权词汇检测模式

### 直接使用商标（高风险）
```
[商标名] + 产品词
例：Nike Headband / Apple Phone Case / Disney Mug
```

### 仿冒暗示（高风险）
```
[商标名] style / [商标名] inspired / [商标名] look
[商标名] quality / [商标名] grade
Fake [商标名] / Replica [商标名] / Knockoff
```

### 虚假正品声称（最高风险）
```
Genuine [商标名] / Authentic [商标名]
Original [商标名] / 100% [商标名]
Official [商标名] / Authorized [商标名]
```

### 混淆性拼写（中高风险）
- 字母替换：Niike / Ad!das / Gooogle
- 空格添加：Air Pods / I Phone
- 连字符：Ap-ple / Sam-sung

## 四、版权侵权

### 产品描述版权

在线内容同样受版权保护，以下行为构成侵权：
- 直接复制竞品 listing 描述（即使做了细微改动）
- 使用他人拍摄的产品图片（未经授权）
- 使用网络上未标注授权许可的图片

**安全做法：**
- 所有产品图片自行拍摄或委托拍摄
- 产品描述原创撰写，不参考竞品文字
- 使用 Unsplash / Pexels 等明确授权的图库图片

### IP 相关图案/图形

以下图形元素在 listing 中使用构成版权/商标侵权：
- 卡通人物形象（米老鼠、皮卡丘等）
- 超级英雄标识
- 品牌 Logo（耐克勾、苹果等）
- 受版权保护的艺术作品再现

## 五、专利声称风险

### 虚假专利声称

在 listing 中声称拥有专利而实际未持有：
- 「Patent Pending」：需已提交专利申请
- 「Patented Design」：需已获授权专利
- 「Exclusive Technology」：需有真实专利支撑

**风险：** FTC 虚假广告诉讼；竞争对手法律诉讼。

### 侵犯他人专利

销售可能侵犯他人专利的产品时，listing 描述不应：
- 强调被专利覆盖的具体技术特征
- 声称该功能为「独创」「首创」
- 提供侵权产品的技术原理说明

## 六、跨境维权现状

### Amazon 渠道
- **品牌投诉（Brand Registry Complaint）**：品牌方直接举报，处理时间 24-72 小时，listing 直接下架
- **亚马逊 Project Zero**：品牌自助删除侵权产品，无需等待亚马逊审核
- **连续侵权**：3 次侵权投诉可能导致账号永久封禁

### 美国法律渠道
- **DMCA Takedown**：版权持有人发出删除通知，需 24 小时内响应
- **商标侵权诉讼**：赔偿范围 $1,000–$200,000/件侵权行为（故意侵权上限 $200 万）
- **ITC 337 调查**：针对进口侵权产品，可申请禁止进口令

### 欧盟渠道
- **EUIPO 投诉**：欧盟知识产权局，可申请边境扣押令
- **海关扣押**：侵权产品在进口时可被海关扣押销毁

## 七、自查清单

在发布 listing 前，逐一确认：

- [ ] 标题中没有使用任何他人注册商标
- [ ] 描述中没有「compatible with」+ 品牌名（或已添加免责声明）
- [ ] 没有使用「genuine / original / authentic」+ 他人品牌名
- [ ] 产品图片没有包含他人 logo 或包装
- [ ] 描述内容是原创，未复制竞品
- [ ] 如声称专利，已实际持有或申请
- [ ] 产品图案/图形不包含受保护的 IP 形象
