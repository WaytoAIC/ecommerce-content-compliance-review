# 跨境合规与产品安全法规参考库

## 一、产品认证要求

### 美国

#### FCC 认证（无线/电子产品）
**适用范围：** 任何含无线发射功能的产品（蓝牙/WiFi/蜂窝/ZigBee）

| 要求类别 | 适用产品 | 合规要求 |
|---------|---------|---------|
| FCC Part 15 | 无意辐射设备（PC/显示器/电源） | SDoC 自我声明或认证 |
| FCC Part 15B | 蓝牙/WiFi 设备 | 需 TCB 第三方认证 |
| FCC Part 22/24/27 | 蜂窝产品 | FCC 认证强制 |

**listing 声称规范：**
```
✅ 「FCC Certified (FCC ID: XXXXXX)」—— 有认证号支撑
✅ 「FCC Compliant」—— 自我声明合规（低风险产品）
❌ 「FCC Approved」—— FCC 不使用「Approved」表述
```

#### UL 认证
- **性质：** 非强制，但零售商（如 Walmart/Target）常要求
- **listing 声称：** 声称「UL Listed」需真实持有证书，否则 🔴 HIGH

#### CPSC（消费品安全委员会）
- **强制标准：** 儿童产品（≤12岁）需符合 CPSC 强制标准
- **CPSIA（2008）：** 儿童产品含铅量 ≤ 100ppm，禁用特定邻苯二甲酸盐
- **违规后果：** 强制召回，罚款 $100,000–$15,000,000

#### FDA 监管产品
| 产品类型 | 监管类型 | listing 注意 |
|---------|---------|------------|
| 膳食补充剂 | DSHEA 备案 | 不得有疾病声称 |
| 医疗器械 | 510(k)/PMA | 声称医疗功效需批准 |
| 化妆品 | 成分报告 | 不得有药品声称 |
| 食品 | 标签法规 | 营养标签强制 |

### 欧盟

#### CE 认证

CE 认证是进入欧盟市场的强制要求（适用于大多数电子/机械/玩具/个人防护设备）。

**声称规范：**
```
✅ 「CE Certified」/ 「CE Marked」（持有 DoC）
✅ 展示 CE 标志图形
❌ 声称「CE Certified」但无技术文件（DoC）→ 🔴 HIGH
❌ 将 CE 误写为「China Export」标志 → 🔴 HIGH
```

**主要 CE 指令：**
| 指令 | 适用范围 |
|------|---------|
| LVD（低电压指令） | 50V-1000V 交流/75V-1500V 直流电气产品 |
| EMC 指令 | 电磁兼容性 |
| RED（无线电设备指令） | 无线设备（取代 R&TTE） |
| GPSD（通用产品安全指令） | 所有消费品 |
| 玩具安全指令 | 14岁以下儿童玩具（EN 71） |
| REACH 法规 | 化学品管制（SVHCs 限制）|
| RoHS 指令 | 电子电气设备有害物质限制 |
| WEEE 指令 | 电子废弃物回收 |

#### GDPR（数据隐私）
- **适用场景：** 智能设备收集用户数据时
- **listing 要求：** 不得虚假声称数据收集范围；需有隐私政策链接

### 英国（脱欧后）

- 使用 **UKCA 标志**（取代 CE，2025 年起强制）
- 部分产品仍可使用 CE 标志（过渡期）
- 通过英国授权代表（UK AR）进行合规

### 日本

#### PSE 认证（电气用品安全法）

**强制认证（不得在日本销售无标志产品）：**

| 分类 | 标志 | 代表产品 |
|------|------|---------|
| 特定电气用品 | ◇（菱形 PSE） | 电源适配器、充电器、电线 |
| 非特定电气用品 | ○（圆形 PSE） | 家用电器、LED 灯、电动工具 |

**listing 声称：**
```
✅ 「PSE認証取得済み」（已获 PSE 认证）
✅ 展示 PSE 标志图片
❌ 声称「PSE 认证」但无实际认证 → 🔴 HIGH（日本法律风险极高）
```

#### 其他日本认证
- **Giteki 标志（技术基准适合证明）：** 无线产品必须
- **VCCI：** 电磁干扰自愿标准（业界通行）

---

### 韩国

#### KC 认证（Korea Certification）

- **强制性：** 电子、电气、儿童产品、安全帽等强制
- 未通过 KC 认证的产品不得在韩国销售
- **listing 声称：** 声称「KC 인증」需真实持有

---

### 澳大利亚

#### RCM 标志
- **强制性：** 电气/电子产品
- 结合 SAA（安全）+ C-Tick（电磁兼容）
- **listing 声称：** 展示 RCM 标志需有真实认证

## 二、进出口管制

### 美国出口管制（EAR）

**受管制产品（不可在 listing 中做出口销售暗示）：**
```
加密软件/硬件（EAR99 以上）
军用级通信设备
高性能计算设备（超出阈值）
夜视/热成像设备
特定半导体/芯片
```

**listing 注意：** 不要在描述中声称产品可出口至美国制裁国家（伊朗/朝鲜/古巴/俄罗斯部分实体）

### ITAR（国际武器交通条例）
- 军事/国防相关产品受 ITAR 管制
- 跨境电商基本不涉及，但无人机/光学设备需注意

### CITES（濒危野生动植物公约）
**严格禁止（🔴 HIGH）：**
- 象牙 / 犀牛角 / 虎骨 / 玳瑁 相关产品
- 珊瑚（活体/制品）
- 特定鸟类羽毛
- 濒危植物衍生品

## 三、危险品运输合规（影响 listing 描述）

### 锂电池（UN38.3）

**listing 必须正确标注：**
```
✅ 电池类型：锂离子/锂聚合物
✅ 电池容量：mAh 或 Wh（不得虚报）
✅ 电池节数
✅ 航空运输限制：≤100Wh（随机携带限制），≥100Wh 需特殊许可

❌ 虚报容量（如实际 3000mAh 标注 10000mAh）→ 🔴 HIGH（欺诈+安全风险）
❌ 声称「航空安全」但不符合 IATA 规定 → 🔴 HIGH
```

### 危险化学品
- 锂电池电解液、特定清洁剂、油漆等受 IATA DG 规范
- listing 不得声称可通过航空快速发货（如实际属于危险品）

## 四、产品标签与标注要求

### 美国标签要求

| 产品类型 | 强制标注内容 |
|---------|-----------|
| 纺织品（FTC Textile Act） | 材质成分（%）、原产国、护理标签 |
| 枕头/床垫 | 填充物成分标签（「Do Not Remove」警告标） |
| 儿童产品 | 追踪标签（制造商/日期/批号）|
| 电器 | 能效标签（Energy Star 等） |
| 食品 | 营养成分表（NFP）、过敏原警告 |
| 药品/OTC | 「Drug Facts」标签 |

**listing 常见问题：**
- 描述材质比例时与实物标签不符 → 🟡 MEDIUM
- 未注明原产国 → 🟡 MEDIUM
- 儿童产品未标注年龄警告 → 🔴 HIGH

### 欧盟标签要求

| 产品类型 | 要求 |
|---------|------|
| 电子产品 | WEEE 标志、CE 标志、制造商/进口商信息 |
| 纺织品 | 纤维成分（多语言）、原产国 |
| 化妆品 | 成分表（INCI 名称）、有效期、制造商 |
| 玩具 | CE 标志、年龄警告、材料 |
| 食品 | 营养标签（多语言）、过敏原 14 大类 |

## 五、特殊品类合规速查

### 无人机（Drone / UAV）

| 市场 | 法规 | listing 要点 |
|------|------|------------|
| 美国 | FAA Part 107 / Remote ID | 250g 以上需注册；声称「FAA Compliant」需准确 |
| 欧盟 | EASA UAS 规范 | C0/C1/C2 分类，标注最大起飞重量 |
| 中国 | CAAC 法规 | 250g 以上实名注册 |

### 激光产品
- FDA CDRH 管辖（美国）
- Class 1–4 分级，Class 3B/4 销售限制严格
- listing 须正确标注激光类别，不得隐瞒

### 隐藏式摄像头 / 监控设备
- 部分州/国家限制销售（如未标注摄像功能的「监控」设备）
- listing 描述需明确说明录像功能，不得模糊

### 防弹/防刺设备
- 美国：多州限制平民购买 III 级以上防弹衣
- listing 需标注合规警告

### 电子烟 / 雾化器
- **Amazon、Shopee、大多数平台明确禁售**
- 不得以「雾化器」「电子设备」等词汇规避审查 → 🔴 HIGH

## 六、合规证书展示规范

在 listing 中引用认证时，正确做法：

```
✅ 「CE Certified – DoC available upon request」
✅ 「FCC ID: 2AXXX-MODEL01」（可查询验证）
✅ 「Meets UL 1642 Standard」（符合标准，非 UL Listed）
✅ 「RoHS Compliant（Certificate available）」

❌ 「CE Certified」（无任何证书信息）→ 虚假声称风险
❌ 「Passed all international certifications」→ 模糊夸大
❌ 「100% safe, certified by all countries」→ 虚假+模糊
```

## 七、原产地声称规范

### 美国 FTC 「Made in USA」标准

**「Made in USA」（无限定语）：**
- 所有或几乎所有主要生产工序在美国完成
- 违规使用 → 🔴 HIGH（FTC 罚款 $46,517/次）

**允许的有限定语声称：**
```
✅ 「Assembled in USA」（在美组装，零件可进口）
✅ 「Designed in USA, Made in China」（准确描述）
✅ 「Made in USA of US and imported parts」
```

### 欧盟原产地
- 欧盟不强制标注原产地（食品除外）
- 声称「Made in Germany / Italy」必须真实
- 「European Quality」模糊声称有争议 → 🟡 MEDIUM
